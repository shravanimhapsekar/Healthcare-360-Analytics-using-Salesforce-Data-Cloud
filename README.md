# Healthcare 360 Analytics using Salesforce Data Cloud
This project involved designing and implementing a Healthcare Analytics Platform using Salesforce Data Cloud and Tableau. 9 healthcare datasets were ingested through Data Streams and transformed into Data Lake Objects (DLOs) and Data Model Objects (DMOs) for unified data management. Identity Resolution, Segmentation and Calculated Insights were created to generate meaningful healthcare metrics and patient-level analytics. The processed data was then connected to Tableau for advanced visualization and reporting. Interactive dashboards were developed to monitor healthcare trends, demographic distributions, patient insights, and performance indicators. The project demonstrates expertise in customer data platforms, data modeling, analytics, and business intelligence.
