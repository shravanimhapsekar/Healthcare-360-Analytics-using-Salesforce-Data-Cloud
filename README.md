# Healthcare 360 Analytics using Salesforce Data Cloud
Designed and implemented a Healthcare Analytics solution using Salesforce Data Cloud and Tableau. Integrated 9 healthcare-related datasets through Data Streams and transformed them into Data Lake Objects (DLOs) and Data Model Objects (DMOs). Configured Identity Resolution (IR) to create unified patient profiles and developed Calculated Insights to derive key healthcare metrics. Built audience Segments for targeted analysis and reporting. Integrated the processed data with Tableau and created interactive dashboards to visualize demographic trends, patient behavior, and healthcare performance indicators. The project demonstrates expertise in customer data platforms, data integration, analytics, and business intelligence.

## Problem Statement
Healthcare organizations often manage patient information across multiple disconnected systems, resulting in fragmented data, duplicate records, and limited visibility into patient behavior and healthcare trends. The objective of this project was to create a unified healthcare data platform using Salesforce Data Cloud that consolidates data from multiple sources, resolves duplicate identities, generates actionable insights, and enables data-driven decision-making through interactive Tableau dashboards.

## Steps Followed
### 1. Data Collection
- Gathered 9 healthcare-related datasets containing demographic and healthcare information.
- Analyzed data structures and relationships across datasets.
### 2. Data Ingestion
- Created Data Streams in Salesforce Data Cloud.
- Loaded datasets into the platform for processing.
### 3. Data Modeling
- Created and configured Data Lake Objects (DLOs).
- Mapped data into Data Model Objects (DMOs).
### 4. Identity Resolution
- Configured Identity Resolution rules.
- Unified duplicate records into a single customer/patient profile.
### 5. Calculated Insights
- Developed business metrics and healthcare KPIs.
- Generated aggregated insights for reporting and analysis.
### 6. Segmentation
- Created audience segments based on demographic and healthcare attributes.
- Enabled targeted analysis of specific patient groups.
### 7. Tableau Integration
- Connected Salesforce Data Cloud with Tableau.
- Prepared analytical datasets for visualization.
### 8. Dashboard Development
- Built interactive dashboards.
- Visualized healthcare trends, demographics, and performance metrics.

## Key Insights
- Created a unified view of patient/customer data across multiple healthcare datasets.
- Reduced data fragmentation through Identity Resolution.
- Generated meaningful healthcare KPIs using Calculated Insights.
- Identified demographic patterns and healthcare trends through segmentation.
- Enabled faster decision-making through interactive Tableau dashboards.
- Improved data accessibility and reporting efficiency using Salesforce Data Cloud.
- Demonstrated end-to-end implementation of a Customer Data Platform (CDP) architecture.
